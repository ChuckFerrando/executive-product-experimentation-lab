# Executive Product Experimentation Lab

## Purpose
This repository demonstrates applied product capability through experiments, prototypes, and AI-assisted builds.

It shows how ideas move from concept to working artifact.

---

## What This Repo Is

- A collection of product experiments and prototypes
- A system for rapid idea validation
- A demonstration of AI-assisted product development
- A proof of execution capability

---

## What This Repo Is NOT

- A production-grade engineering repository
- A collection of random side projects
- A code-only repository without context
- A showcase without structured thinking

---

## Who This Is For

- Executive product leaders
- Builders and technologists
- Hiring managers evaluating execution capability

---

## How to Use This Repo

- Start with `START_HERE.md`
- Review `REPO_MAP.md`
- Explore experiments
- Follow build logs and learnings
- Review reusable patterns and workflows
