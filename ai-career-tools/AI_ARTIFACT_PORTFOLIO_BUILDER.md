# AI Artifact Portfolio Builder

## Problem

AI hiring is increasingly artifact-driven, but most candidates lack structured ways to build, organize, and present meaningful artifacts.

## Hypothesis

If candidates are guided through structured artifact creation aligned to real hiring signals, their employability will increase significantly.

## Concept

A system that helps users:

- identify high-value AI skills
- build artifacts aligned to those skills
- structure and present those artifacts as a portfolio

## Core Features

- Artifact templates (evaluation systems, prompts, workflows)
- Guided build paths (12-week plans)
- Portfolio assembly system
- Skill-to-artifact mapping

## Architecture (Conceptual)

- Input: user skill level + target role
- Engine: artifact recommendation + build guidance
- Output: structured portfolio of artifacts

## Initial Build Idea

- Simple web app or Notion-based system
- Pre-built templates + guided workflows
- Example artifacts included

## Success Criteria

- Users produce multiple high-quality artifacts
- Artifacts map directly to hiring signals
- Improved interview conversion rates

## Next Steps

1. Define 5–10 canonical artifact types
2. Build first template set
3. Test with small user group
4. Iterate based on outcomes
